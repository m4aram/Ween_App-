# Ween 

A smart restaurant discovery app that helps users find the best restaurants based on their food cravings.

## Features

- 🤖 **Smart Chat** — Type a dish and get the best restaurant recommendation instantly
- 🗺️ **Google Maps Integration** — Open restaurant location directly in Google Maps
- 📸 **Photo Gallery** — Browse restaurant photos before you go
- 📋 **Menu Viewer** — View the menu or open the restaurant's website
- ❤️ **Favorites** — Save your favorite restaurants
- 🌍 **Multi-language** — Supports Arabic and English
- 🔍 **City Filter** — Filter restaurants by city
- 📶 **Offline Detection** — Notifies you when there's no internet connection

## Tech Stack

- **Flutter** — Cross-platform mobile development
- **Firebase Firestore** — Cloud database
- **Firebase Authentication** — Google Sign-In
- **GetX** — State management & navigation
- **url_launcher** — Open links and maps
- **connectivity_plus** — Network detection
